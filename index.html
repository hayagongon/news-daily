💡 PATの作成方法：
GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens
→ "news-daily" リポジトリを選択 → Contents: Read and Write → Generate
除外キーワード（1行1キーワード）
記事タイトルにこのキーワードが含まれる記事を非表示にします。
説明文はチェックしないので、企業記事で「円安が追い風」と書かれていても除外されません。
具体的なフレーズほど誤除外が少なくなります（例：「円安」より「円安が進」）。
🤖 AIフィルタールール
Claude AI がルールを読み取り、毎晩の記事取得時に高度な除外判定を行います。
変更後は翌晩の取得から反映されます（キーワード除外より正確な判定が可能です）。
